Консольная программа для загрузки музыки из сервиса **Яндекс Музыка**.
С её помощью вы можете сохранять треки напрямую по ссылкам на конкретные композиции, альбомы или даже страницы исполнителей.
Автоматически сортирует по папкам и заполняет метаданные для каждого аудиофайла: название, исполнитель, альбом, год выпуска и т. д.
А ещё добавляет обложку альбома к каждому файлу!

## Установка и настройка

### 1. Клонирование репозитория
Клонируйте репозиторий с **GitHub** в текущую директорию:
```bash
git clone https://github.com/MaxMavr/YandexMusicDonwloader.git
```

### 2. Установка зависимостей
Убедитесь, что у вас установлен **Python** и необходимые библиотеки:
- `yandex-music` — для работы с API Яндекс Музыки
- `requests` — для HTTP-запросов, используется внутри yandex-music и для обложек
- `mutagen` — для работы с метаданными MP3 файлов
- `python-dotenv` — для загрузки переменных из `.env` файла


### 3. Настройка файла `.env`
В репозитории присутствует файл `env` — это шаблон для создания файла `.env`.
Создайте файл `.env` в корневой директории проекта, далее необходимо заполнить две обязательные переменные:

- Ваш OAuth-токен от **Яндекс Музыки**:
```env
YANDEX_TOKEN=ВАШ ТОКЕН ЯНДЕКС
```
Его можно получить [по этой инструкции](https://yandex-music.readthedocs.io/en/main/token.html).

- Путь к папке, куда будут скачиваться треки:
```env
DOWNLOAD_PATH=C:\...\путь\до\скачанных\треков
```


### 4. Создание ярлыка для запуска
Создайте ярлык с следующей структурой команды:
```env
C:\Windows\System32\cmd.exe /c "C:\...\путь\до\main.bat"
```

## Использование

### Запуск через ярлык
Просто дважды щелкните по созданному ярлыку для запуска программы. Вы можете закрепить этот ярлык или даже поменять его иконку.

### Запуск вручную
Вы также можете запустить программу напрямую через:
- `main.bat` — **Windows**
- `main.py` — через **Python**

## Послесловие
- Убедитесь, что у вас есть активная подписка _Яндекс Плюс_ для доступа к трекам
- Скачивание защищенного контента может быть ограничено правами правообладателей
- Используйте программу только в личных целях в соответствии с условиями использования **Яндекс Музыки**

При возникновении проблем обращайтесь, всегда рад поправить свой кодикс <._.>
